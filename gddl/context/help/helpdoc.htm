权限设置

1． 权限组设置

权限组的用途：当某一岗位有多人操作时，设置权限组可以简化系统操作，当需修改权限时，只需修改权限组，即可修改全部工作人员的权限。

每个单位可根据工作需要设置若干各权限组。


设置步骤：点击[权限管理] 进入权限组设置页面 → 点击[添加]填加新一行  → 在新添加栏填写组名（如“日报”）→在描述栏填写对此组的任务 → 点击[保存]按纽 保存数据。


→ 再选择新加组 → 点击[设置权限]按纽 → 显示见图：→ 选择权限资源，如图   → 点击[保存]按纽 → 点击[返回]按纽完成权限组设置

 

2．  人员信息和权限设置


点击[人员信息]按纽 → 进入人员信息界面 → 点击[添加]按纽 → 添加新一行（见图）


→ 输入用户名（用名称第一字母）→ 输入姓名（用汉字） → 输入部门（厂内二级单位名称）→ 输入职位（岗位名称）→ 选择是否登陆（是可以登陆，否则不能登陆，可用此对此人进行登陆限制）→ 输入工作电话，便于有问题时联络 → 点击[保存]按纽，保存数据。

3． 初始密码与密码修改

初始密码与用户名相同，用户登陆后请先更改密码。

更改密码：→ 点击[修改人员信息] → 进入人员信息修改界面，输入原密码 → 输入新密码 → 点击[修改密码] → 密码修改完成。

4． 密码重置

由于工作人员疏忽，密码丢失和遗忘，可以对密码进行重置：

→ 点击[人员信息]按纽 → 选择需重置密码的人员行 → 点击[重置密码] → 密码重置，密码同用户名相同。
